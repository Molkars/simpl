
# SimpL
> **Simp**le **L**anguage

## Installation

```console
cargo build --release
cp ./target/release/simplc /usr/bin/simplc
```

## Usage

```console
simplc main.simp
./main
```

## Project Management

simpl comes with a built in project management solution: it's self.

Just define a `build.simp` file to manage compilation, linking, and program features
